# xhycom

**xhycom** reads HYCOM model output in the native `.a/.b` binary format directly into [xarray](https://docs.xarray.dev) Datasets — no conversion to NetCDF required.

It is inspired by [xmitgcm](https://xmitgcm.readthedocs.io), which does the same thing for MITgcm output.
